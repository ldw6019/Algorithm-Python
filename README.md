# 🏆 Algorithm Solutions
> Python을 이용한 백준(BOJ) 문제 풀이 기록입니다.

## 📌 Directory Structure
* **그래프**: 최단 경로, 최소 신장 트리 등
* **그리디 알고리즘**: 탐욕법을 이용한 최적해 구하기
* **BFS와 DFS**: 너비/깊이 우선 탐색 활용
* **브루트포스**: 완전 탐색을 통한 문제 해결
* **스택과 큐 / 정렬과 탐색**: 필수 자료구조 및 알고리즘
* **수학**: 정수론 및 기하학 관련
* **미분류**: 분류 예정 및 기초 문제

---

## 🛠 Naming Convention
* 파일명 규칙: `BOJ_[문제번호]_[문제명]_[티어].py`
* 예시: `BOJ_7576_토마토_G5.py`
